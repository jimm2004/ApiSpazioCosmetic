# 💄 API Spazio Cosmetic

Bienvenido al repositorio oficial del backend de Spazio Cosmetic (MoodStore).

> **Aviso:** Esta es una actualización en modo prueba para verificar el flujo de trabajo de Git (Commit y Push) y el despliegue automático en producción. 🚀

## 📝 Sobre el Proyecto

Esta es una API construida con **Laravel** que sirve como el núcleo central para la gestión de Spazio Cosmetic. Proporciona todos los servicios necesarios para alimentar la aplicación móvil desarrollada en Flutter.

### Características Principales:
* **Gestión de Catálogo e Inventario:** Control total sobre los productos importados.
* **Autenticación de Usuarios:** Sistema seguro de login y gestión de credenciales.
* **Módulo de Cobranza:** Funciones específicas para el rol de "cobrador", incluyendo el registro de visitas a clientes y seguimiento de pagos.
* **Gestión de Colecciones:** Organización estructurada del catálogo.

## 🛠️ Tecnologías Utilizadas

* **Framework:** Laravel (PHP)
* **Base de Datos:** MySQL
* **Entorno de Desarrollo Local:** Laragon
* **Producción:** Hostinger
